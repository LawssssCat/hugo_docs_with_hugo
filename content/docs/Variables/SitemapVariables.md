# Sitemap Variables - Sitemap变量 

[https://gohugo.io/variables/sitemap/](https://gohugo.io/variables/sitemap/)



​	sitemap是一个 `Page` ，因此具有可用于sitemap模板的所有[页面变量](https://gohugo.io/variables/page/)。 它们还具有以下专门用于sitemap的变量：

### .Sitemap.ChangeFreq

​	页面更改频率 

### .Sitemap.Priority

​	页面的优先级 

### .Sitemap.Filename

​	Sitemap文件名
