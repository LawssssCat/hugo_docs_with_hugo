# uniq

[https://gohugo.io/functions/uniq/](https://gohugo.io/functions/uniq/)

Takes in a slice or array and returns a slice with duplicate elements removed.

## 语法

```
uniq SET
{{ slice 1 3 2 1 | uniq }} --> [1 3 2]
```
