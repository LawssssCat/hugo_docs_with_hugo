# strings.Repeat

[https://gohugo.io/functions/strings.repeat/](https://gohugo.io/functions/strings.repeat/)

Returns INPUT repeated COUNT times.

## 语法

```
strings.Repeat COUNT INPUT
{{ strings.Repeat 3 "yo" }} → "yoyoyo"
{{ "yo" | strings.Repeat 3 }} → "yoyoyo"
```
